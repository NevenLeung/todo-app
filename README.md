# todo-app

一个使用原生js写的待办事项应用:memo:

## 功能

### 基础功能

- [x] todo的展示
- [x] todo的输入、添加
- [ ] todo的状态toggle（未完成、已完成）
- [ ] 点击已添加的todo进入可编辑状态
- [ ] 删除已添加的todo
- [ ] 按todo的状态作展示（全部、未完成、已完成）

### 可强化功能

- [ ] 选择截止时间
- [ ] 给todo设置背景颜色，作视觉的分类
- [ ] 使用鼠标拖拽，调整todo的排列顺序
- [ ] 父子todo，可设置层级关系
- [ ] 设置分类（项目）对todo进行分组，项目的添加、删除、编辑、排序
- [ ] 拖拽todo加入分类（项目）

### 强化体验功能

- [ ] 在todo中使用特定符号及相应文本，为todo添加不同的属性（所属项目、优先级、标签）
- [ ] 快捷键
- [ ] 右键菜单
- [ ] 全文搜索
- [ ] 标签
- [ ] 智能识别日期
- [ ] 项目名称补全
- [ ] 过滤器
- [ ] 评论系统
- [ ] 提醒功能
- [ ] 合作项目中的指派todo负责人
